Regex Cli

Posted: 2023-07-22   |   Tags: c++, cli, java, linux

Simple command-line program that lets you specify a file and a regular expression pattern to match against in that file. Implemented in C++ and in Java (separately). Java version showcases the Strategy Pattern, supports additional features like verbose mode and line mode, and has documentation. Shipped (C++) executable for Linux, Java version requires a runtime.
